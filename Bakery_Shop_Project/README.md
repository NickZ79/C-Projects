# üçû Bakery Shop Order Form

The Bakery Shop Order Form is a Windows Forms application created for a C# class assignment. It simulates a simple checkout process for a bakery, allowing the user to select items, calculate totals, and reset the form.

---

## ‚ú® Features

- Select bakery items and quantities  
- Calculate:
  - Subtotal  
  - Sales tax  
  - Optional discount  
  - Final order total  
- Clear/reset all fields to start a new order  
- Basic input validation  

---

## üßÆ How It Works

1. The user selects items and enters quantities.  
2. When **Calculate** is clicked, the program computes:
   - Item subtotal  
   - Sales tax  
   - Discount (if used)  
   - Grand total  
3. When **Clear** is clicked, the form resets to its default state.
